Welcome to SLogo!

SLogo stands for Simple Logo and is an integrated development environment (IDE) that will allow you, the user, to interact with a virtual turtle that is capable of moving and drawing.

When using this program, type your commands into the text field on the bottom of the screen. You may run anything from a single command to a loop of commands as well. Heck, you can even save commands by using the respective button on the side bar!
